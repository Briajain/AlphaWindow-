**AlphaWindow** is a high-density, local-first intelligence platform designed for deep research and market analysis. It transforms raw social fragments from X into a structured, AI-augmented knowledge base.

---

## 🏛️ System Architecture

AlphaWindow operates as a unified quad-layered intelligence stack:

1.  **Ingestion (Chrome Extension)**: High-speed scraper that "captures" raw data from your browser.
2.  **Processing (Express Backend)**: Authenticated gateway that manages data integrity and SQL operations.
3.  **Intelligence (Ollama + Qwen 2.5)**: Local AI pipeline that performs category classification, sentiment analysis, and confidence scoring without external API calls.
4.  **Storage (PostgreSQL)**: Industrial-grade archival of every intel fragment, note, and user identity.

---

## 🚀 Core Features

### 1. Research Intelligence Feed
A real-time "Alpha Dashboard" for your data. Every captured post is automatically enriched with:
- **Category Tags**: (Finance, Defense, Tech, Energy, Macro, etc.)
- **Sentiment Spectrum**: (Bullish / Bearish / Neutral)
- **Advanced Sorting**: Instantly re-order the live feed by **Timestamp**, **Views**, or **Likes** to identify high-impact signals.
- **Confidence Scoring**: A 0-100% reliability index generated by the local AI analyst.

### 2. The Multi-Model Vault
A knowledge management suite where you can:
- Store parsed research notes.
- Track long-term hypotheses.
- Link fragments to specific projects.

### 3. Integrated DBMS Console
A tactical SQL driver built directly into the dashboard.
- **Schema Explorer**: View all tables and row counts in real-time.
- **Raw Execution**: Run any PostgreSQL command directly from the web terminal.
- **Quick-Strike Queries**: Pre-loaded commands for "Top Tickers" and "Volume Analysis."

### 4. AI Research Analyst (Multi-Source RAG)
A dedicated research interface where you can query your entire database using natural language.
- **Social Media Grounding**: Synthesizes live X fragments to report on discussion volume and sentiment.
- **Private Intel Repository**: Upload **PDF** and **Markdown** documents to the node. The AI will integrate their content into its analysis for deep-dive reporting.
- **Tactical Synthesis**: Strictly grounded in your private fragments and documents to prevent hallucinations.

## 🛠️ Installation & Setup (Portability Protocol)

AlphaWindow is designed for "Clone & Deploy" portability. Follow these two steps on any new system:

### 1. Initialize System
Ensure you have **Node.js**, **PostgreSQL**, and **Ollama** installed.
```bash
npm install
npm run setup
```
The `setup` command automatically configures your `.env`, initializes the PostgreSQL database, and pulls the required AI models.

### 2. Launch Terminal
```bash
npm run dev
```
Access your dashboard at `http://localhost:3000`.

---

## 🩺 System Health (Doctor Mode)
If you encounter connectivity issues on a new PC, run the diagnostic tool:
```bash
npm run doctor
```
This provides a "Green/Red" report on your Node, Database, and Intelligence status.

## 🛡️ Security Protocol
- **Zero-Telemetery**: All AI analysis happens on your local VRAM.
- **JWT Protection**: The DBMS and Vault are guarded by local session tokens.
- **PostgreSQL Isolation**: Your database is hosted locally and never exposed to external networks.

> [!TIP]
> **Performance Mode**: For maximum analysis speed, ensure your GPU has at least 4GB of VRAM available for the Ollama inference pipeline.
